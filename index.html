MalCare

This project aims to study files with malware. Studying the malware files include finding out about their contents as well the family it belongs to. The major plus point of our method is that we plan to classify this data without actually running/ opening the files. This is done by using image processing techniques to classify the malware into families. Binaries of the malware file are first converted into grayscale images. These converted images are then classified using certain similarities in their image texture. We explored various machine learning algorithms for the classification and made use of a sequential convolutional neural network. This neural network will help us classify the data based on different image features. We also added new converted malware files to our already existing database of malware images. This new dataset also will help the community and further research in this area.
